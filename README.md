# CVM Push Generation

Сервис для автоматической генерации текстов push-уведомлений для акций ДИКСИ на основе данных из Google Sheets.

## Возможности

- 📋 Просмотр акций из вкладки "CVM offline" с фильтрацией
- ✨ AI-генерация нескольких вариантов текстов push для каждой акции
- 📅 Настройка количества push-отправок, дат и времени
- 📝 Редактируемые правила генерации текстов
- ✏️ Редактирование сгенерированных текстов с подсчётом символов
- 👁️ Предпросмотр push-уведомления
- 💾 Сохранение результатов обратно в Google Sheets (вкладка "PUSH")

## Быстрый старт

### 1. Установка зависимостей

```bash
cd cvm_push_generation
pip3 install -r requirements.txt
```

### 2. Настройка

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

**Минимально необходимо:**
- API ключ AI-провайдера (Anthropic или OpenAI) — вводится в интерфейсе
- Google Sheets таблица должна быть открыта для чтения по ссылке

**Для сохранения в Google Sheets** (опционально):
- Создайте Service Account в Google Cloud Console
- Дайте ему доступ к таблице (Editor)
- Скачайте JSON-ключ и загрузите через интерфейс приложения

### 3. Запуск

```bash
streamlit run app.py --server.port 8502
```

Приложение будет доступно по адресу: http://localhost:8502

## Как использовать

1. **Введите API ключ** в боковой панели (Anthropic или OpenAI)
2. Перейдите на вкладку **"Акции"** — просмотрите доступные промо-акции
3. Перейдите на **"Правила генерации"** — настройте стиль и тон текстов
4. На вкладке **"Генерация PUSH"**:
   - Выберите акцию
   - Задайте количество push-отправок и расписание
   - Нажмите "Сгенерировать"
   - Отредактируйте тексты при необходимости
   - Выберите лучшие варианты
   - Сохраните в Google Sheets

## Технологии

- Python 3.11 + Streamlit
- Google Sheets API (gspread)
- Claude API (Anthropic) / OpenAI API
