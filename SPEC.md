# CVM Push Generation — Спецификация

## Назначение
Сервис для автоматической генерации текстов push-уведомлений на основе данных об акциях из Google Sheets (вкладка "CVM offline"), с возможностью редактирования и сохранения результатов обратно в таблицу (вкладка "PUSH").

## Функциональные требования

### 1. Загрузка данных
- Чтение акций из вкладки "CVM offline" (только строки с Каналы коммуникации = "PUSH")
- Чтение существующих push из вкладки "PUSH"
- Фильтрация по месяцу/неделе/статусу

### 2. Конфигурация генерации
- Пользователь задаёт количество push-сообщений на акцию
- Пользователь задаёт даты и время отправки для каждого push
- Пользователь задаёт/редактирует правила генерации текстов (промпт)
- Ограничения на длину заголовка и текста (настраиваемые)

### 3. AI-генерация текстов
- Поддержка Claude API (Anthropic) и OpenAI API
- Генерация нескольких вариантов заголовка и текста для каждого push
- Учёт: названия акции, описания, механики, сегмента, категории, сроков
- Соблюдение ограничений по длине символов
- Стиль и тон задаются через настраиваемые правила

### 4. Редактирование
- Просмотр сгенерированных вариантов в удобном интерфейсе
- Выбор лучшего варианта или ручное редактирование
- Счётчик символов в реальном времени
- Предпросмотр push-уведомления

### 5. Сохранение в Google Sheets
- Запись отредактированных push обратно во вкладку "PUSH"
- Обновление существующих строк или добавление новых
- Сохранение всех полей: сегмент, дата, время, заголовок, текст, ссылка

## Архитектура

### Технологический стек
- **Python 3.11** + **Streamlit** (веб-интерфейс)
- **gspread** + **google-auth** (Google Sheets API)
- **anthropic** / **openai** (AI генерация)
- **Порт: 8502** (не конфликтует со стандартными сервисами)

### Структура проекта
```
cvm_push_generation/
├── app.py                  # Главный файл Streamlit-приложения
├── config.py               # Конфигурация (ID таблицы, настройки)
├── sheets_client.py        # Работа с Google Sheets API
├── ai_generator.py         # Генерация текстов через AI
├── requirements.txt        # Зависимости
├── .env.example            # Пример переменных окружения
├── .gitignore              # Исключения для git
├── SPEC.md                 # Эта спецификация
├── README.md               # Инструкция по запуску
└── credentials/            # (не в git) Google API credentials
    └── service_account.json
```

### Авторизация Google Sheets
- Service Account с доступом к таблице (для чтения/записи)
- Файл credentials хранится локально, не в git

### Переменные окружения (.env)
```
AI_PROVIDER=anthropic          # или openai
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_CREDENTIALS_PATH=credentials/service_account.json
SPREADSHEET_ID=1-jsqs-YChB9uN56PcQ2aWqR01MW3O-uaTunNKYZJ7IY
PORT=8502
```

## Запуск
```bash
pip install -r requirements.txt
streamlit run app.py --server.port 8502
```
